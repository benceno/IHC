Hospital Accompaniment Feature

The following images illustrate a user connecting with a hospital iPad for virtual accompaniment during a doctors visit. The user first selects "Connect With Hospital iPad" on their DigiFam device. This gives a call ID and informs the user to get help from an employee. From here a hospital employee will use the DigiFam app on their device to set up the call. All they have to do is enter the call ID and this will connect the call.
